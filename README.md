# Текстовий парсер на Java (Лабораторна робота №5)

## Опис проєкту
Ця програма призначена для декомпозиції тексту на складові частини (речення, слова, символи) та виконання маніпуляцій з ними. Проєкт демонструє навички роботи з об'єктно-орієнтованим дизайном та регулярними виразами в Java.

## Основний функціонал
* **Парсинг тексту:** Розбиття вхідного рядка на масиви об'єктів (Речення -> Слова -> Символи).
* **Опрацювання тексту:** Програма знаходить у кожному реченні слова заданої довжини та замінює їх на вказаний підрядок.
* **Обробка помилок:** Реалізовано виловлювання виняткових ситуацій (наприклад, порожній вхідний текст).

## Технічний стек
* **Мова:** Java 
* **Методи:** Регулярні вирази (Regex), StringBuilder для оптимізації роботи з пам'яттю, принципи ООП.
